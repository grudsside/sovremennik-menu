# Загрузка на GitHub Pages

1. Распакуйте архив.
2. Откройте папку `sovremennik-service-supabase`.
3. Загрузите в GitHub не саму папку, а все файлы и папки внутри нее.
4. В корне репозитория должны быть `index.html`, `assets`, `data`, `supabase`, `.nojekyll`.
5. После загрузки откройте сайт и войдите под `grigory / 0808`.
