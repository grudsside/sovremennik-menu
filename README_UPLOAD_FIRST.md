# Быстрая загрузка на GitHub Pages

1. Распакуйте архив `sovremennik-menu-github-pages-fixed.zip`.
2. Откройте папку `sovremennik-menu-github-pages-fixed`.
3. В GitHub загрузите именно содержимое этой папки, а не саму папку и не ZIP.
4. В корне репозитория должны быть видны: `index.html`, `data`, `assets`, `README.md`, `.nojekyll`.
5. Включите Pages: `Settings → Pages → Deploy from a branch → main → / root → Save`.
6. Открывайте ссылку вида `https://ВАШ_ЛОГИН.github.io/sovremennik-menu/`.

Если виден только заголовок `sovremennik-menu`, скорее всего, вы открыли страницу репозитория, а не опубликованный сайт GitHub Pages.
