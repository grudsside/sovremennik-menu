# Approved interface v3

Дата согласования: 2026-07-18.

## Интерфейс

- боковая навигация сгруппирована на «Основное», «Работа» и «Управление»;
- «Мои задачи» вынесены в отдельный основной раздел;
- главная показывает актуальные задачи, сотрудников сегодняшней смены и ближайшие события из расписания;
- быстрый доступ и отдельная кнопка истории уведомлений не используются;
- история уведомлений открывается колокольчиком;
- расписание отображается месячной сеткой;
- форма сообщения об ошибке содержит тип «Другое»;
- значок настроек в верхней панели удаляется;
- мобильное меню открывается отдельной кнопкой, закрывается после выбора раздела, по фону и Escape.

## Безопасность изменений

Основная бизнес-логика `app.js`, Supabase, уведомления, авторизация, чек-листы и ревизии не изменяются. Версия подключается отдельными файлами `interface-v3.css` и `interface-v3.js` поверх ранее опубликованного интерфейса.
