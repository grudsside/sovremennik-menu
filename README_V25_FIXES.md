# V25 — правки сотрудников, тех. карт и методички

## Обязательно перед проверкой новых функций

В Supabase выполните файл:

`supabase/sql/STEP_8_CONTENT_EDITING_METHOD_TECH.sql`

Затем обновите Edge Function для сотрудников:

```powershell
supabase functions deploy admin-employees --use-api
```

## Что проверить на сайте

1. Главная страница: появилась карточка с пометкой `БЕТА-ВЕРСИЯ`.
2. Сотрудники: кнопка `Удалить` должна деактивировать сотрудника и обновить список.
3. Тех. карты: одна кнопка `Редактировать тех. карты`; внутри окна есть добавление, редактирование и удаление.
4. Методичка: кнопка `Редактировать позиции`; внутри окна есть добавление, редактирование и удаление карточек.
5. После загрузки файлов на GitHub Pages сделайте `Ctrl + Shift + R`.
