# Современник — база сотрудников

Сайт для GitHub Pages. Внутри есть три основных раздела:

1. **Методичка** — теория, бар, кухня, десерты и архив.
2. **Чек-листы** — открытие/закрытие, заготовки, генеральная уборка.
3. **Тех. карты** — технологические карты напитков и заготовок бара.

## Как загрузить на GitHub Pages

1. Распакуйте архив.
2. Загрузите **содержимое папки** в корень репозитория, а не саму папку целиком.
3. В корне должны лежать: `index.html`, папки `assets`, `data`, файл `.nojekyll`.
4. В настройках GitHub Pages выберите `main` и `/ root`.

## Как редактировать карточки методички

Основные данные находятся в `data/menu.json`.

## Как редактировать чек-листы и тех. карты

На сайте показана текущая HTML-копия из загруженных Excel-файлов. Оригиналы лежат в папке:

`assets/documents/`

Если нужно обновить чек-лист или тех. карту, замените соответствующий Excel-файл и обновите `data/menu.json`, либо пришлите новый файл для пересборки сайта.
