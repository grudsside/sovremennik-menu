# Современник · сервис сотрудников

В этой версии добавлены:

- все чек-листы свернуты по умолчанию и открываются по нажатию;
- отправка доступна для всех чек-листов;
- после отправки показывается окно: «Отлично! Чек-Лист отправлен»;
- пользователь остается в текущей вкладке, перехода в «Контроль» нет;
- добавлен раздел «Ревизии» с формой ежедневной ревизии кофе;
- в «Контроль» добавлены две папки: «Чек-листы» и «Ревизии»;
- данные чек-листов и ревизии кофе отправляются в Google Sheets через Apps Script.

## Важно: обновите Apps Script

Чтобы ревизия кофе записывалась в лист `Ревизия Кофе`, нужно заменить код в Google Apps Script на код из файла:

```text
google_apps_script_code.js
```

После замены кода нужно заново нажать:

```text
Deploy → Manage deployments → Edit → New version → Deploy
```

Ссылка Web App может остаться прежней. Она уже подключена в `assets/js/app.js`.

## Загрузка на GitHub Pages

Распакуйте архив и загрузите содержимое папки в корень репозитория. В корне должны лежать:

```text
index.html
assets/
data/
README.md
.nojekyll
google_apps_script_code.js
```

Если GitHub Pages уже включен, дополнительных настроек обычно не нужно.
