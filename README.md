# Современник — методичка для сотрудников

Готовый статичный сайт для GitHub Pages.

## Что внутри

- `index.html` — главная страница сайта.
- `data/menu.json` — база карточек и обучающих материалов.
- `assets/css/styles.css` — стили сайта.
- `assets/js/app.js` — логика вкладок, поиска и раскрывающихся блоков.
- `assets/images/` — фотографии блюд/десертов и схемы для раздела «Теория».
- `.nojekyll` — служебный файл для GitHub Pages.

## Основные разделы

- Теория
- Бар
- Кухня
- Десерты
- Архив

## Как опубликовать на GitHub Pages

1. Создать репозиторий, например `sovremennik-menu`.
2. Распаковать архив.
3. Загрузить в репозиторий именно содержимое папки, а не сам ZIP.
4. В корне репозитория должны лежать `index.html`, папки `assets`, `data`, файл `README.md` и `.nojekyll`.
5. Открыть `Settings → Pages`.
6. Выбрать `Deploy from a branch`.
7. Branch: `main`, folder: `/ root`.
8. Сохранить и дождаться публикации.

## Как редактировать карточки

Основные карточки находятся в `data/menu.json` в массиве `items`.

Обучающие разделы находятся в `data/menu.json` в массиве `lessons`.

После правки файла нажать `Commit changes`. Ссылка сайта останется прежней.

## Как добавить новое обучение

Можно взять шаблон `data/template_new_lesson.json`, скопировать блок в массив `lessons` внутри `data/menu.json` и заполнить поля.

Поддерживаемые типы блоков в обучении:

- `lead` — вводный текст;
- `image` — картинка или схема;
- `cards` — набор небольших карточек;
- `steps` — нумерованный список действий;
- `table` — таблица;
- `checklist` — чек-лист;
- `callout` — важная заметка.

## Как добавить фото

Фото кладутся в папки:

- `assets/images/bar/`
- `assets/images/kitchen/`
- `assets/images/desserts/`
- `assets/images/archive/`
- `assets/images/theory/`

В карточке или обучении нужно указать путь, например:

```json
"image": "assets/images/kitchen/shakshuka.jpg"
```

## Важно

В `index.html` также есть встроенная резервная копия `menu.json`, чтобы сайт открывался локально. На GitHub Pages основным источником будет `data/menu.json`.
