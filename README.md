# Современник — сервис для интеграции сотрудников

Готовая версия сайта для GitHub Pages.

## Что внутри

- `index.html` — главная страница сайта.
- `data/menu.json` — база карточек, чек-листов, тех. карт и обучений.
- `assets/css/styles.css` — стили сайта.
- `assets/js/app.js` — логика сайта.
- `assets/images/` — изображения.
- `assets/documents/` — Excel-файлы чек-листов и тех. карт.
- `.nojekyll` — служебный файл для GitHub Pages.

## Подключение Google Sheets

В этой версии раздел **Контроль** подключен к Google Apps Script:

```text
https://script.google.com/macros/s/AKfycbxsT5RXCzV6GVjpYU0DFDMWmM4vQR5t03JumsOb-hdNhtaWL7e6K4G2C9XE1cFYy-nM/exec
```

Отправка чек-листов открытия и закрытия уходит в Google Таблицу через Apps Script. Раздел **Контроль** загружает записи из этой общей таблицы, поэтому данные видны со всех устройств.

## Как загрузить на GitHub

1. Распакуйте архив.
2. Откройте папку `sovremennik-service-github-pages-sheets`.
3. Загрузите в корень репозитория **содержимое папки**, а не саму папку целиком.
4. В корне репозитория должны лежать:
   - `index.html`
   - `data/`
   - `assets/`
   - `README.md`
   - `.nojekyll`
5. GitHub Pages должен быть настроен на `main / root`.

## Как проверить работу контроля

1. Откройте сайт по ссылке GitHub Pages.
2. Перейдите в раздел **Чек-листы**.
3. Откройте чек-лист открытия или закрытия.
4. Введите имя сотрудника.
5. Отметьте пункты и нажмите **Отправить**.
6. Перейдите в раздел **Контроль** и нажмите **Обновить данные**.

Если записи не появились, проверьте в Apps Script, что web app опубликован с доступом **Anyone** и выполняется от имени владельца скрипта.
